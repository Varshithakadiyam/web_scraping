# web_scraping
Amazon Laptop Data Scraping using Python Project Overview This project uses Python web scraping techniques to collect laptop product information from Amazon. It extracts important details such as product title, price, brand, RAM, storage, rating, operating system, color, and processor. The collected data can be used for analysis, comparison, and research purposes.
